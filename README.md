# Install / Publish

typically lucally `$PREFIX='/build/'`

```
$ PREFIX='/'
$ npm install
$ git clone https://github.com/phosphorjs/phosphorjs.github.io build
$ node index.js $PREFIX
$ cd build
$ git add *
$ git commit -am'build docs'
$ git push origin master:master
```
